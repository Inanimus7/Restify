
Το παρόν project ακολουθεί όλες τις απαιτούμενες τακτικές που τέθηκαν από το σεμινάριο.
Πέρα από τις βασικές προδιαγραφές, ενσωματώθηκαν επιπλέον λειτουργίες (extra features) 
με σκοπό να βελτιώσουν την εμπειρία χρήστη και να επεκτείνουν τις δυνατότητες του project.
Ενώ το ζητούμενο design ήταν αρκετά απλό, επέλεξα να δημιουργήσω μια πιο σύνθετη εκδοχή του. 
Αυτό έγινε αρχικά για πειραματισμό και δοκιμή διαφορετικών προσεγγίσεων, 
αλλά και για να αξιολογήσω την ικανότητά μου να υλοποιήσω ένα πιο σύνθετο design.
